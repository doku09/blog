# git add [파일명]
특정파일 staged 상태에 add하기
```
git add [파일명]
```
현재 디렉터리 밑에 있는 모든 파일을 add
```
git add .
```
디렉토리 밑에 있는 모든 파일 add
```
git add [디렉토리 이름]
```
-a는 add의 약자
add와 커밋이 동시에 된다.  
```
git commit -am
```
# git reset - git add 취소하기
add된 파일 취소
```

```
특정 파일 add 취소
```
git reset HEAD [파일명]
```

# git branch
현재 브랜치를 조회한다.
```

```


# git checkout
해당 브랜치로 이동
```
git checkout [브랜치명]
```
새로운 브랜치 생성 후 동시에 체크아웃
```
git checkout -b
```
